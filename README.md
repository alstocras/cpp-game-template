# c++ game template
this is a small template to quickstart making c++ games with raylib. this is not affiliated with raylib *at all*, and its been made for my use, although you may use it. You do need to separately install raylib. try looking at your system's package manager.
## how to contribute
always use PRs and use the following guidelines
- `patch:` for small fixes
- `feature:` for new features
- `release:` for software-breaking changes
### make your PRs small
Please make them easy to review and commit small changes.
## that's it!
thanks for contributing!
